[Nest](https://github.com/nestjs/nest) 框架TypeScript启动库。

## 安装

```bash
$ npm install
```

## 运行

```bash
# development开发
$ npm run start

# watch mode热重载
$ npm run start:dev

# production mode生产模式
$ npm run start:prod
```
